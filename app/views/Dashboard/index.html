Nos projets 2011 - 2012

Pour la rentrée le laboratoire java vous propose déjà quelque projets, qui conviendront à tous les niveaux.

Parmi ces projets il y'a pour commencer Goodquestion, l'idée est très simple, une personne pose une question avec trois réponses possible pour la soumettre au vote des internautes. Ensuite lorsque l'internaute répond à la question, goodquestion return le pourcentage de réponse au choix proposé.

Pour les fan de probabilités, le laboratoire java vous propose de développer une application Android, nom de code PP.
